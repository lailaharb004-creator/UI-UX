# 🥗 Daily UI #004 – Calorie Calculator

A clean and modern **Calorie Calculator** mobile UI designed for the **Daily UI Challenge #004 – Calculation**.

This concept helps users estimate their daily calorie needs based on personal information such as gender, age, height, weight, activity level, and fitness goal. The interface focuses on simplicity, readability, and an intuitive user experience.

---

## ✨ Challenge

**Daily UI #004 – Calculation**

> Design a calculation element or interface. It can be a standard calculator, a scientific calculator, or a specialty calculator for something such as calorie intake, mortgage payments, credit scores, or any other calculation.

---

## 🎨 Design Concept

The design consists of two mobile screens:

- **Welcome Screen** introducing the calorie planning experience.
- **Calculator Screen** where users enter their information and calculate their recommended calorie intake.

The interface uses a minimal layout with a soft green color palette to create a healthy and calming feel.

---

## 📱 Screens

### Welcome Screen
- App introduction
- Motivational headline
- Call-to-action button
- Sign Up navigation

### Calculator Screen
- Gender selection
- Age input
- Height input
- Weight input
- Daily activity level slider
- Fitness goal selection
- Calculate button

---

## 🎯 Design Goals

- Create an intuitive calorie calculator interface.
- Design a clean and easy-to-use mobile experience.
- Organize user inputs with a clear visual hierarchy.
- Maintain consistency in spacing, typography, and components.
- Use colors that reflect health and wellness.
