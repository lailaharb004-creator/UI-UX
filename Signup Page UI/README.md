# 🦖 Daily UI #001 – Dinosaur Museum Event Sign Up

A modern and responsive **Sign Up UI** created for the **Daily UI Challenge #001**.

This design is for a fictional **Dinosaur Museum Event**, allowing users to register for an interactive dinosaur exhibition. The project focuses on creating a clean, user-friendly registration experience while maintaining a playful visual style.

---

## ✨ Challenge

**Daily UI #001 – Sign Up**

> Create a sign up page, modal, form, or app screen related to signing up for something. It could be for a volunteer event, contest registration, a giveaway, or anything you imagine.

---

## 🎨 Design Concept

The goal was to design an engaging registration page for a dinosaur museum event that appeals to families and dinosaur enthusiasts.

The interface combines:

- Clean and minimal layout
- Friendly illustration
- Soft blue color palette
- Responsive desktop and mobile versions
- Simple and intuitive form design

---

## 📱 Screens

### Desktop
- Hero section with event information
- Event illustration
- Sign Up / Log In toggle
- Email & Password fields
- Remember Me option
- Forgot Password link
- Social sign in buttons

### Mobile
- Responsive version of the desktop design
- Optimized spacing and layout
- Touch-friendly interface
